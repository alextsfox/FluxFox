# FluxFox: a Python Package for Processing Eddy Flux data
Currently, this package only supports post-processing methods for eddy flux data, but in the future will include preprocessing methods as well.

## Features
- U* filtering of eddy flux data
- Retrieval of Ameriflux data

## Planned
- Preprocessing functions for eddy flux data
- Flux computation
- QA/QC functions
- Gap-filling functions


## Requirements
* pandas
* numpy
* requests
* solarpy
* tqdm (can change)
* openpyxl
